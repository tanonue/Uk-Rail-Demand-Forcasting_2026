# Uk-Rail-Demand-Forcasting-2026
Time-series forecasting of UK rail passenger demand to support capacity and revenue planning.

#Project Overview
This project analyses UK rail passenger demand and builds a time-series forcasting model to support capaity and revenue planning.

## Data Source
Passenger rail usage data from the UK government office of Rail and Road (Open government license).

## Tools
- Python
- Pandas, matplotlib
- Power BI (for reporting)

## Project Status
Ongoing : Data loading and exploratory analysis.
